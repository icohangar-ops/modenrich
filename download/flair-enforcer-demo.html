ML Classification Engine

TF-IDF inspired scoring — pure TypeScript, no external APIs

Step 1 — Tokenize
Strip punctuation, lowercase, split on whitespace, filter 80+ stop-words
Step 2 — Build Index
Weighted term index from keyword rules + learned training data
Step 3 — Score
Score each flair: sum(matched weights) / sqrt(token count)
Step 4 — Rank
Confidence = top_score / (top_score + second_score)

Example Classification

Post: "Need help with Python asyncio"
↓
Tokens: [need, help, python, asyncio]
↓
Matched: "python" → Programming Help, "help" → Help, "asyncio" → Programming Help
↓
Result: "Programming Help" — confidence: 0.87

Length normalization: sqrt dampening prevents longer posts from dominating. A 500-word essay won't outrank a precise 20-word question.
